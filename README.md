# Mandelbrot-zoom
Zooming into the Mandelbrot Set.
Not finished yet.

## Installation

You need to be [FLTK](https://www.fltk.org/) installed on your machine.

### Linux

Clone & cd :
```
git clone https://github.com/Nicolas-Reyland/Mandelbrot-zoom
cd Mandelbrot-zoom
```
Compile :
```
./compile
```
Usage :
```
./mandelbrot
```
